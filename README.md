# dakoku

## 事前条件

- [Cloud SDK](https://cloud.google.com/sdk/?hl=ja)
- [Java8](http://www.oracle.com/technetwork/java/javase/overview/index.html)

## ビルド

```sh
./gradlew build
```

## ローカルで実行

```sh
./gradlew appengineRun
```
を実行して http://localhost:8080/hello をブラウザで開く。
